# Python

Create your virtual environment

```bash
python -m venv NAME_OF_VENV
```

Activate venv

```bash
# https://docs.python.org/3/library/venv.html#how-venvs-work
```

Deactivate venv

```bash
deactivate
```

Install dependencies in venv

```bash
poetry install
```

Start your project

```bash
poetry run start
```
